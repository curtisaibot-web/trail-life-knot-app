# Knot IQ (The Bear Essentials) - Forensic Deconstruction

## 1. Core Value Proposition
*   **The BEST 3D Knot Tying Resource:** Fully interactive, 3D, and offline-capable knot-tying educational tool.
*   **Expert-Led:** Backed by "The Bear Essentials" outdoors brand, focusing on practical bushcraft and survival skills.
*   **Utility & Convenience:** Designed for field use with offline access and customizable lists.

## 2. Target User Profiles
*   **Outdoors Enthusiasts:** Campers, hikers, and bushcrafters needing reliable field skills.
*   **Professionals:** Firefighters, Arborists, Search and Rescue (SAR) personnel.
*   **Hobbyists:** Sailors, fishers, and climbers.
*   **Youth & Educators:** Scouts and Trail Life members (supported by family sharing).

## 3. Primary Use Cases
*   **In-Field Reference:** Checking a knot while off-grid (offline mode).
*   **Skill Acquisition:** Learning a new knot from scratch using 3D step-by-step animations.
*   **Expert Mastery:** Refining advanced techniques for professional or survival scenarios.
*   **Instructional Tool:** Leaders teaching knots to a group using 3D visuals.

## 4. Feature Set
*   **Interactive 3D Animations:** 360° rotation, zoom, and manual scrubbing (playthrough slider).
*   **Offline Access:** Core functionality available without internet.
*   **Comprehensive Library:** 185+ knots with expert instructions.
*   **Visual Customization:** Rope color switches and Left-Handed mode.
*   **Utility Tools:** Quick search, category filters, and customizable "Favorites" lists.
*   **Educational Content:** Pros/Cons comparisons, real-world use cases, and video tutorials.

## 5. User Journey (Inferred)
1.  **Open:** App launches to a categorized home screen (Camping, Bushcraft, etc.).
2.  **Search/Browse:** User selects a category or searches for a specific knot (e.g., "Bowline").
3.  **View:** User enters the 3D viewer.
4.  **Interact:** User rotates the knot, scrubs the animation slider, and switches to left-handed mode if needed.
5.  **Learn:** User reads the description, pros/cons, and real-world applications.
6.  **Exit:** User saves the knot to a custom list or closes the app after mastering the tie.

## 6. UI/UX Structure
*   **Home Screen:** Grid-based category layout with "Recently Viewed" section.
*   **Navigation Bar:** Bottom nav for Home, Search, Favorites, and Settings.
*   **Knot Viewer:** Large 3D viewport with playback controls (Play/Pause, Slider, Speed) and customization toggles (3D Freeform, 90° rotate, Reset).
*   **Info Panel:** Collapsible or scrollable section for text-based instructions and use cases.

## 7. Content Structure
*   **Categorization:** Grouped by activity (Climbing, Fishing, Boating, Firefighting, etc.).
*   **Metadata:** Each knot includes name, difficulty, category, description, pros/cons, and use cases.

## 8. Learning Model
*   **Visual-First:** Heavy reliance on 3D spatial understanding.
*   **Manual Control:** Encourages the user to "scrub" the animation to see the exact crossover points.
*   **Contextual:** Explains *why* to use a knot, not just *how*.

## 9. Retention Mechanics
*   **Favorites/Custom Lists:** Encourages users to build their own "toolbox."
*   **Continuous Content:** Regular updates to the knot library.
*   **Brand Ecosystem:** Integration with Bear Essentials gear and YouTube community.

## 10. Monetization Strategy
*   **One-Time Purchase:** $6.99 CAD (Paid App model).
*   **Family Sharing:** Value-add for families on iOS.
*   **Ecosystem Driver:** Serves as a gateway to the Bear Essentials physical gear shop.
