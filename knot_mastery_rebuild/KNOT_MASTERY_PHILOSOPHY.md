# KNOT MASTERY — Practical Skill System

## 1. CORE PHILOSOPHY: Skill-Building Platform
Knot Mastery is not a digital book; it is a **digital coach**. It moves the user from passive observation to active mastery through a closed-loop system of instruction, practice, and verification.

## 2. THE LEARNING SYSTEM (The Mastery Loop)
*   **LEARN:** High-fidelity 3D animation with "Trace Path" (glowing trail) and voice narration.
*   **PRACTICE:** "Ghost Mode" where the user follows the animation in real-time with a built-in timer.
*   **TEST:** A timed challenge where the animation is hidden, and the user must tie the knot from memory.
*   **MASTER:** User submits a photo/video (AI-verified or peer-reviewed) to earn the "Mastery" seal for that knot.

## 3. THE USE-CASE ENGINE (Contextual Problem Solving)
Instead of searching for "Bowline," users can search for "Scenarios":
*   **"Secure a boat to a dock"** → Recommends: Cleat Hitch, Bowline.
*   **"Tighten a tent guy-line"** → Recommends: Taut-Line Hitch, Midshipman's Hitch.
*   **"Rescue someone from a hole"** → Recommends: Bowline on a Bight.
*   **"Bundle firewood"** → Recommends: Timber Hitch, Constrictor Knot.

## 4. THE PRACTICE & CHALLENGE SYSTEM
*   **Drill Mode:** Repeat a specific knot 10 times with a decreasing timer to build muscle memory.
*   **The "Blackout" Challenge:** Tie the knot with the screen off (simulating night/blind tying).
*   **Troop Duels:** Real-time timed challenges against other users (perfect for Trail Life).

## 5. THE PROGRESSION SYSTEM (Identity-Based Growth)
Users earn XP and Ranks that reflect their real-world capability:
*   **Level 1: Beginner (The Tenderfoot)** - Knows the basic 6 survival knots.
*   **Level 2: Outdoorsman** - Master of hitches and lashings.
*   **Level 3: Operator** - Fast, reliable tying in high-stress scenarios.
*   **Level 4: Survivalist (The Master)** - Can tie any knot in the library blindfolded and under time pressure.

## 6. RETENTION MECHANICS (The "Addictive" Engine)
*   **Streaks:** Daily "One Knot" challenge to keep the skill sharp.
*   **The Progress Tree:** A visual forest that grows and thrives as the user masters more knots.
*   **Skill Decay:** If a knot isn't practiced for 30 days, its "Mastery" icon fades, prompting a quick review.
