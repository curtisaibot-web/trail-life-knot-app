# KNOT MASTERY - Lovable Build Prompts

Copy and paste these prompts into Lovable to build the core system.

### **Prompt 1: The Woodsy Dashboard & Progress Tree**
"Build a mobile-first dashboard for a knot-tying app called 'Knot Mastery.' The UI should have a 'woodsy' and 'old-school video game' vibe with a color palette of forest green (#2D5016), earthy brown (#8B6F47), and trail red (#C41E3A). Feature a 'Progress Tree' at the top—a visual illustration of a tree that grows as the user earns XP. Include a 'Daily Streak' counter with a fire emoji and a 'Next Knot to Master' card that shows the user's current target knot with its name and a 'Learn' button."

### **Prompt 2: The Scenario-Based Recommendation Engine**
"Create a 'Scenario Finder' screen. At the top, place a search bar with the placeholder 'What do you need to do? (e.g., Secure a tent, Rescue)'. Below the search, show a list of filter chips for categories like 'Camping,' 'Sailing,' 'Survival,' and 'Utility.' When a user selects a scenario or category, display a list of recommended knots. Each knot card should show the knot name, a difficulty badge (Beginner, Intermediate, Advanced), a short one-sentence use-case description, and an 'Open Skill Lab' button."

### **Prompt 3: The Skill Lab (3D Viewer Interface)**
"Design a 'Skill Lab' screen for learning a specific knot. The top half should be a large, dark viewport for a 3D model (use a placeholder image/div for now). Below the viewport, add a 'Playthrough Slider' to scrub through an animation from 0% to 100%. Include playback controls: Play/Pause, Reset, and a speed selector (0.5x, 1x, 2x). Add a toggle for 'Left-Handed Mode' and a 'Trace Path' button. At the bottom, list the step-by-step instructions in a clear, numbered list."

### **Prompt 4: The Practice & Mastery System**
"Build a 'Practice Mode' overlay for the Skill Lab. When activated, it should start a stopwatch timer at the top. Display the current step instruction prominently. Add a large 'I Tied It!' button. When clicked, stop the timer, show a 'Well Done!' animation, calculate XP earned, and update the user's 'Best Time' for this knot. If it's a new record, show a trophy icon."

### **Prompt 5: The Verification & Testing Portal**
"Create a 'Knot Test' screen. This screen should hide the step-by-step instructions and the 3D animation. The user is prompted to 'Tie the [Knot Name] from memory.' Include a camera preview window where the user can take a photo of their finished knot. Add a 'Submit for Verification' button that sends the photo to a (placeholder) verification service and shows a 'Pending' status."

### **Prompt 6: The Troop Leaderboard**
"Design a social leaderboard screen. It should have two tabs: 'My Troop' and 'Global.' Display a ranked list of users with their rank number, avatar, username, total XP, and current streak. Highlight the current user's row in trail blue (#003DA5). Add a section at the bottom showing 'Troop Stats' like 'Total Knots Tied Today' and 'Active Missions.'"
